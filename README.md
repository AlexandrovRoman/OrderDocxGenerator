| Variable        | Short description | Default |

| TEMPLATES_FOLDER      | Папка с шаблонами |   templates |
| CONTRACT_NUMBER | Номер договора | 17052020 |
| CONTRACT_SIGN_AT | Дата подписания (%d.%m.%Y) | 17.05.2020 |
